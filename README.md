# Мы поставили себе цель автоматизировать процесс оплаты питания и прохода в столовую. Наш проект будет состоять из трех частей: телеграмм бота для пользователя, приложения для администратора и приложения для проверяющего на входе в столовую.

# Телеграмм бот будет отвечать за общение с пользователем, а именно: 
# -регистрировать его в системе, 
# -настраивать дни, в которые он хочет обедать или завтракать, 
# -выдавать квитанцию для оплаты, 
# -добавлять в базу данных запрос на проверку платежа,
# -показывать талончик для прохода в столовую.

# Приложение для администратора предназначается для:
# -проверки платежей из базы данных, составленной телеграмм ботом, 
# -просмотра главной базы данных со списками всем обедающих и завтракающих учеников,
# -просмотра некоторых других полезных сведений.

# Приложение для проверяющего будет использовано для:
# -считывания талончика, который выдаётся пользователю телеграмм ботом, 
# -его проверки на подлинность,
# -получение из него данных об учащимся.

# Для получения талончика пользователю нужно:
# 1) найти нашего бота и прислать ему свои фио и класс 
# !у каждого учащегося должен быть свой телеграмм аккаунт, т.к. они будут связаны в базе данных!
# 2) указать дни недели, в которые он хочет обедать и/или завтракать (также можно указать конкретные дни)
# 3) оплатить питание по квитанции, присланной ботом
# 4) дождаться подтверждения от администратора 
# 5) на входе в столовую запросить у бота талончик и показать проверяющему

# Порядок проверки платежей администратором:
# 1) открыть в приложении вкладку проверки платежей
# 2) каждый платеж, указанный в приложении найти в списке переводов и платежей на банковский счёт
# 3) также у каждого платежа сверить суммы указанные в приложении и в информации по платежу
# 4) отметить в приложении все правильно совершенные платежи
# 5) закончить проверку, нажав кнопку "подтвердить"

# Проверка талончика на входе в столовую:

# 2) проверить, что показывает приложение 
# (фио+класс в случае подлинности; красный крестик в случае подделки или же отсутствия оплаты питания)
